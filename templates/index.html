Turn customer feedback into decisions.

InsightLoop is a small, self-hosted web app that analyzes reviews, support tickets and survey comments with any LLM. You get sentiment, topics, urgency, and a live dashboard.

Try the demo → View on GitHub
🔍

Structured analysis

Sentiment, score 0–100, topics, urgency 1–5, summary, and concrete next steps — for every feedback item.

📊

Live dashboard

KPIs, sentiment distribution, top topics, and a 14-day trend line update as new feedback arrives.

🧠

Provider-agnostic

Swap between OpenAI, Anthropic, a local Ollama model, or the built-in Mock — with a single environment variable.

Get started in 60 seconds

No API key required to try — the Mock provider gives you a fully working demo out of the box.

git clone https://github.com/RainCherb/insightloop.git
cd insightloop
python -m venv .venv
.\.venv\Scripts\Activate.ps1
pip install -r requirements.txt
Copy-Item .env.example .env
python main.py
# open http://localhost:8000

✅ Single binary of code — no microservices, no Kafka, no Redis

✅ REST API + HTML UI — embed it in your stack or use it standalone

✅ Tests + CI — pytest + GitHub Actions included
